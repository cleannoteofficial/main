# 클린노트 웹사이트 수정 가이드

---

## 0. GitHub 접속하기 (첫 번째 단계)

### Step 1: GitHub.com 열기

1. 웹 브라우저 주소창에 `github.com` 입력
2. 엔터 키로 이동

### Step 2: 로그인하기

1. 화면 오른쪽 위의 **Sign in** 버튼 클릭
2. 구글 아이디(이메일)와 비밀번호 입력
3. **Sign in** 클릭

### Step 3: 저장소(Repository) 찾기

1. 로그인 후 company-website-final 클릭

### Step 4: 파일 수정하기

1. 수정하고 싶은 파일 클릭 (예: `config.js`)
2. 파일 우측의 ✏️ **연필 아이콘** 클릭
3. 코드 수정
4. 아래의 **Commit changes** 클릭
5. 메시지 작성 후 **Commit** 클릭

## 핵심 요약: 어떤 파일을 수정하나요?

| 수정하고 싶은 것 | 수정할 파일 | 난이도 |
|:---|:---|:---|
| 전화번호, 이메일, 주소, 회사명 | `config.js` | 쉬움 |
| 서비스 가격 | `config.js` | 쉬움 |
| 부가서비스 가격 | `config.js` | 쉬움 |
| 포트폴리오/SNS 링크 | `config.js` | 쉬움 |
| 고객 리뷰 추가/삭제/수정 | `content.js` | 보통 |
| 히어로 문구, 통계 숫자 | `content.js` | 쉬움 |
| 회사 소개 문구 | `content.js` | 쉬움 |
| **서비스 카드 수정/추가/삭제** | `content.js` | 보통 |
| **서비스 진행 과정 수정** | `content.js` | 쉬움 |

> 주의: `site-loader.js`, `script.js`, `quote.js`, CSS 파일, HTML 파일은 **절대 수정하지 마세요!**

## 1. 전화번호 변경하기

### 파일: `config.js`

파일을 열고 아래 부분을 찾으세요:

```javascript
    // 대표 전화번호 (네비게이션 바, 푸터에 표시)
    phone: '0000-0000',
```

따옴표 안의 `0000-0000` 을 새 번호로 변경하세요:

```javascript
    phone: '02-1234-5678',
```

### 견적 페이지 상담 전화번호도 별도로 있습니다:

```javascript
    // 상담 전화번호 (견적 페이지 오른쪽에 표시)
    consultPhone: '1234-5678',
```

→ 원하는 번호로 변경:

```javascript
    consultPhone: '02-9876-5432',
```

## 3. 서비스 가격 변경하기

### 파일: `config.js`

`pricing:` 섹션에서 원하는 서비스를 찾으세요.

**예시: 입주 청소 가격 변경**

현재:
```javascript
        // 입주 청소
        ipju: {
            basePrice: 150000,
            displayPrice: '150,000원~',
            cardPrice: '15만원~',
            priceNote: '20평 기준',
        },
```

가격을 200,000원으로 올리려면:
```javascript
        // 입주 청소
        ipju: {
            basePrice: 200000,
            displayPrice: '200,000원~',
            cardPrice: '20만원~',
            priceNote: '20평 기준',
        },
```

### 각 항목 설명:

| 항목 | 설명 | 예시 |
|:---|:---|:---|
| `basePrice` | 견적 계산에 사용되는 기본 가격 (숫자만) | `200000` |
| `displayPrice` | 견적 페이지 가격표에 표시 | `'200,000원~'` |
| `cardPrice` | 메인 페이지 서비스 카드에 표시 | `'20만원~'` |
| `priceNote` | 가격 기준 설명 | `'20평 기준'` |

> `basePrice`는 따옴표 없이 숫자만! 나머지는 따옴표('') 안에!

## 5. 면적별 가격 배수 변경하기

### 파일: `config.js`

```javascript
    areaMultipliers: {
        '10-20': 1.0,    // 10-20평: 기본가격 그대로
        '20-30': 1.3,    // 20-30평: 기본가격 × 1.3
        '30-40': 1.6,    // 30-40평: 기본가격 × 1.6
        '40-50': 2.0,    // 40-50평: 기본가격 × 2.0
        '50+':   2.5,    // 50평 이상: 기본가격 × 2.5
    },
```

**계산 방법:**
- 입주 청소(15만원) + 20-30평(×1.3) = 195,000원
- 입주 청소(15만원) + 40-50평(×2.0) = 300,000원

**예시: 30-40평 배수를 1.8로 변경:**
```javascript
        '30-40': 1.8,    // 30-40평: 기본가격 × 1.8
```

## 7. 고객 리뷰 추가하기

### 파일: `content.js`

### 새 리뷰 추가

`reviews: [` 바로 아래에 다음 형식을 복사해서 붙여넣으세요:

```javascript
        {
            name: '홍**',
            initial: '홍',
            service: '입주청소',
            rating: 5,
            text: '여기에 리뷰 내용을 작성하세요.',
            date: '2026년 3월 1일',
        },
```

### 각 항목 설명:

| 항목 | 설명 | 예시 |
|:---|:---|:---|
| `name` | 고객 이름 (보통 성+**) | `'홍**'` |
| `initial` | 프로필에 표시될 한 글자 | `'홍'` |
| `service` | 이용한 서비스 이름 | `'입주청소'` |
| `rating` | 별점 (1~5, 숫자만) | `5` |
| `text` | 리뷰 본문 내용 | `'정말 좋았습니다!'` |
| `date` | 리뷰 날짜 | `'2026년 3월 1일'` |

### 리뷰 삭제

삭제하고 싶은 리뷰의 `{ }` 블록 전체를 삭제하세요.

**예시: 김** 리뷰 삭제 전:**
```javascript
    reviews: [
        {
            name: '김**',
            initial: '김',
            service: '입주청소',
            rating: 5,
            text: '이사 후 입주청소를 맡겼는데...',
            date: '2026년 1월 28일',
        },
        {
            name: '이**',
```

**삭제 후:**
```javascript
    reviews: [
        {
            name: '이**',
```

## 서비스 카드 수정하기 (서비스 페이지)

### 파일: `content.js`

서비스 페이지에 표시되는 각 서비스 카드의 내용을 수정할 수 있습니다.

### 카드 내용 수정하기

`content.js`에서 `serviceCards` 배열을 찾으세요. 각 카드는 아래와 같은 구조입니다:

```javascript
    {
        id: 'ipju-cleaning',                          // ← 고유 ID (수정하지 마세요)
        icon: 'service-card-icons/입주청소.png',       // ← 아이콘 이미지 파일
        title: '입주 청소',                             // ← 카드 제목
        subtitle: '깨끗한 집, 행복한 가족',              // ← 소제목
        description: '거실, 침실 등 가정 전체를...',     // ← 설명
        time: '3-5시간',                                // ← 소요 시간
        team: '2-3명',                                  // ← 투입 인원
        price: '10만원~',                               // ← 표시 가격
        includes: [                                     // ← 포함 사항 목록
            '바닥 및 벽면 먼지·오염 제거',
            '주방 기름때 및 싱크대·수납장 내부 청소',
            ...
        ],
    },
```

→ 변경 예시 (가격과 소요 시간 수정):
```javascript
        time: '4-6시간',
        price: '15만원~',
```

### 새 서비스 카드 추가하기

`serviceCards` 배열의 맨 끝에 아래 형식을 복사해서 붙여넣기 하세요:

```javascript
        {                                              // ← 마지막 카드의 }, 뒤에 추가
            id: 'new-cleaning',
            icon: 'service-card-icons/아이콘파일명.png',
            title: '새 서비스 이름',
            subtitle: '소제목을 입력하세요',
            description: '서비스에 대한 설명을 입력하세요.',
            time: '3-5시간',
            team: '2-3명',
            price: '10만원~',
            includes: [
                '포함 사항 1',
                '포함 사항 2',
                '포함 사항 3',
                '포함 사항 4',
                '포함 사항 5',
            ],
        },
```

> 아이콘 이미지 파일은 `service-card-icons` 폴더에 넣어두세요.
> 포함 사항(includes)은 원하는 만큼 추가하거나 줄일 수 있습니다.

### 서비스 카드 삭제하기

삭제하고 싶은 카드의 `{ }` 블록 전체를 삭제하세요.

예시 - 아래 부분을 통째로 삭제:
```javascript
        {                           // ← 여기부터
            id: 'special-cleaning',
            icon: '...',
            title: '특수 청소',
            ...
        },                          // ← 여기까지 삭제
```

### 카드 순서 바꾸기

원하는 카드의 `{ }` 블록 전체를 잘라내기(Ctrl+X) 후 원하는 위치에 붙여넣기(Ctrl+V) 하세요.

## 서비스 진행 과정 수정하기

### 파일: `content.js`

서비스 페이지 하단의 4단계 프로세스를 수정할 수 있습니다.

```javascript
    processSteps: [
        { iconType: 'phone',    number: 'STEP 01', title: '상담 신청',   description: '전화 또는 온라인으로...' },
        { iconType: 'estimate', number: 'STEP 02', title: '맞춤 견적',   description: '공간 크기와 상태에...' },
        { iconType: 'calendar', number: 'STEP 03', title: '예약 및 방문', description: '원하시는 날짜와...' },
        { iconType: 'thumbsup', number: 'STEP 04', title: '청소 완료',   description: '꼼꼼한 청소 후...' },
    ],
```

→ 변경 예시:
```javascript
        { iconType: 'phone', number: 'STEP 01', title: '무료 상담', description: '카카오톡 또는 전화로 상담하세요.' },
```

> `iconType`은 다음 4가지만 사용 가능합니다: `'phone'`, `'estimate'`, `'calendar'`, `'thumbsup'`

## 자주 하는 실수와 해결법

### 실수 1: 따옴표를 빼먹음

```javascript
// 잘못된 예
phone: 02-1234-5678,

// 올바른 예
phone: '02-1234-5678',
```

### 실수 2: 쉼표를 빠뜨림

```javascript
// 잘못된 예 (쉼표 없음)
phone: '02-1234-5678'
email: 'test@test.com',

// 올바른 예 (쉼표 있음)
phone: '02-1234-5678',
email: 'test@test.com',
```

### 실수 3: 중괄호/대괄호 삭제

```javascript
// 잘못된 예
pricing:
    ipju:
        basePrice: 150000,

// 올바른 예
pricing: {
    ipju: {
        basePrice: 150000,
    },
},
```

### 실수 4: 숫자에 따옴표를 넣음

```javascript
// 잘못된 예
basePrice: '150000',

// 올바른 예
basePrice: 150000,
```

### 실수 5: 숫자에 쉼표를 넣음

```javascript
// 잘못된 예
basePrice: 150,000,

// 올바른 예
basePrice: 150000,
```

## 문제 해결 (FAQ)

### Q: 수정했는데 웹사이트에 반영이 안 돼요

1. 파일을 저장했는지 확인 (Ctrl + S)
2. 브라우저에서 강력 새로고침 (Ctrl + Shift + R)
3. 브라우저의 캐시 삭제 (Ctrl + Shift + Delete)

### Q: 수정 후 웹사이트가 안 나와요 / 깨졌어요

1. 브라우저에서 F12 → Console 탭 → 빨간색 오류 메시지 확인
2. 대부분 따옴표, 쉼표, 중괄호 문제입니다
3. 백업 폴더에서 원본 파일을 복사해서 복원하세요

### Q: 어떤 파일도 수정하지 않았는데 문제가 생겼어요

인터넷 연결을 확인하세요 (Google Fonts를 로드합니다)
